# MiniCube2

What is this ? 

An 8 bit fantasy computer. utilizing a 6502 CPU running 32768 cycles per frame, roughly 2mhz CPU.

64kb of memory

a display of 64x64 8 bit pixels.

featuring 6 channel audio, and a blitter chip for fast memory transfer.
